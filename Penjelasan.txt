1. github
2. gitlab

keduanya mempunya kelebihan dan kekurangan.

github sangat mudah untuk digunakan dengan adanya github desktop

gitlab tidak memiliki versi desktop harus by commandline untuk updatenya.
kelebihannya adalah pada mode private gratis.

sampai saat ini saya sedang mendalami gitlab
